========================================
  Daily_Checklist_Machine - Langkah di Server IIS
========================================

1. Copy seluruh isi folder INI ke server
   (misal: C:\inetpub\wwwroot\Daily_Checklist_Machine)

2. Di IIS Manager:
   - Application Pools -> Add -> Name: Daily_Checklist_Machine
   - .NET CLR version: No Managed Code -> OK
   - Sites -> Add Website -> Site name: Daily_Checklist_Machine
   - Physical path: (folder yang kamu copy)
   - Application pool: Daily_Checklist_Machine
   - Binding: http, port 80 (atau yang lain) -> OK

3. Klik kanan folder aplikasi -> Properties -> Security
   -> Edit -> Add -> ketik IIS_IUSRS -> beri Read & Execute

4. Jika connection string beda di server, edit
   appsettings.json (atau buat appsettings.Production.json)

5. Pastikan di server sudah terpasang:
   .NET 8 Hosting Bundle
   https://dotnet.microsoft.com/download/dotnet/8.0

6. Start website di IIS.

Panduan lengkap: lihat DEPLOY_IIS.md di source project.
